# Changelog

All notable changes to the Pawa AI VS Code Extension will be documented in this file.

## [1.0.0] - 2025-01-04

### Added
- Initial release of Pawa AI VS Code Extension
- AI Chat panel with streaming responses
- Context-aware code assistance
- Code generation command
- Explain code command
- Refactor code command
- Fix bug command
- Add comments command
- Generate tests command
- Right-click context menu integration
- Keyboard shortcuts (Ctrl+Shift+P for chat, Ctrl+Shift+G for generate)
- Diff preview before applying changes
- Persistent conversation history
- Beautiful chat UI with markdown support
- Code block syntax highlighting
- Configuration options for API URL, model, temperature, etc.
- Automatic context building from current editor
- Support for multiple programming languages

### Features
- Real-time streaming AI responses
- Claude Code-like integration
- Works with self-hosted Pawa AI backend
- No cloud dependency
- Free and open source

### Technical
- TypeScript implementation
- Webview-based chat interface
- Axios for HTTP communication
- VS Code Extension API integration
- Comprehensive error handling
- Type-safe codebase
