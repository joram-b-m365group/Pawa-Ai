# Point of Sale System
This is a basic Point of Sale (POS) system built with Python and Tkinter.

## Features
* Product management: add products with name, price, and quantity
* Sales tracking: make sales and view sales history
* Total calculation: calculates the total amount of all sales

## Usage
1. Run the `pos.py` file to start the POS system.
2. Add products by filling in the product name, price, and quantity, and clicking the "Add Product" button.
3. Make sales by selecting a product, entering the quantity, and clicking the "Confirm Sale" button.
4. View sales history in the sales listbox.
5. The total amount of all sales is displayed at the bottom of the sales frame.
